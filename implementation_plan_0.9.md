# Implementation Plan - Archon 0.9 (Design Freeze)

## Goal
Freeze the design scope for cure/heal rules before implementation begins.

## Design Decisions (To be finalized in 0.9)
- Determine if cure happens through a 'heal' spell.
- Determine if cure is combat-only or board-level.
- Determine if cure removes imprisonment only or also restores HP.

## Out of Scope
- Full mana system
- Full spell targeting system
- AI implementation
- Major UI redesign
- Broad content expansion
