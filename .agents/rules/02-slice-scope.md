# Part 1 game slice scope

Build one combat slice only.

Required for the slice:
- one board or arena context
- 2 light units
- 2 dark units
- minimal combat HUD
- one projectile or beam path
- one hit / impact path
- one status path
- one barrier or dodge path
- one death path
- music + SFX + minimal voice cue support

Do not build the full game.
Prove the import contract and one slice.
