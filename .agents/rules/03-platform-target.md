# Platform target

For Part 1, the combat slice is a **web browser build** using the existing Vite + React stack.

Do not re-platform to React Native / Expo during Part 1.

Reason:
- the Workshop already runs on the web stack
- export and review flow are already browser-oriented
- the fastest proof is Workshop -> export -> browser combat slice

A mobile/native decision belongs to Part 2 after the export contract and the first combat slice are stable.
