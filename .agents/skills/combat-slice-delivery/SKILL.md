# Combat Slice Delivery

Use this skill for the first playable Archon combat slice.

## Slice target
A minimal but real scene that proves:
- exported assets load
- units render
- HUD renders
- VFX trigger in context
- audio can be triggered
- the slice remains driven by approved exports

## Required behavior
- keep runtime simple
- keep the slice in the web Vite/React stack for Part 1
- do not invent permanent asset ids
- rely on the export contract
- leave the slice easy to expand in Part 2

## Output contract
Return:
- files changed
- importer changes
- slice behavior added
- proof that approved exported assets are used
